# Stock Analyzer – Financial Analytics Dashboard
## Personal Project | Python + Streamlit

An **interactive web-based financial analytics platform** built from scratch, featuring quantitative risk modeling, volatility estimation, and real-time equity analysis.

---

## Features

- **Price Trend Visualization** – Historical price charts with interactive zoom
- **Moving Averages** – MA20 and MA50 overlays for trend identification
- **GARCH Volatility Modeling** – Time-varying volatility estimation using GARCH(1,1)
- **Value-at-Risk (VaR)** – Historical VaR at 95% and 99% confidence levels
- **Interactive Dashboard** – Built with Streamlit for real-time user interaction

---

## Tech Stack

`Python` `Streamlit` `arch` `yfinance` `pandas` `numpy` `matplotlib` `plotly`

---

## Running the App

```bash
pip install -r requirements.txt
streamlit run app.py
```
